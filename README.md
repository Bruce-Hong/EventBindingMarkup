EventBindingMarkup
==================

Markup Extensions for Events (WPF 4.5)

![](src/Images/xaml.png)
## Command
 1. {eb:EventBinding}  (Simple naming pattern to find Command)
 1. {eb:EventBinding Command=CommandName}

## CommandParameter
 1. $e (EventAgrs)
 2. $this or $this.Property
 3. string
